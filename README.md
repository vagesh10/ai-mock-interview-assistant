\# 🎯 AI Mock Interview Assistant



An AI-powered mock interview platform that simulates real-world technical interviews using voice interaction, coding challenges, and intelligent feedback.



This project helps users practice interviews in a realistic environment by combining conversational AI, code execution, and performance analysis.



---



\## 🚀 Features



\* 🤖 AI-generated interview questions using Gemini API

\* 🎤 Voice-based interaction for realistic interview simulation

\* 💻 Built-in code editor for solving coding problems

\* 📊 Automated scoring and feedback system

\* 🕘 Interview history tracking

\* 🔐 Secure authentication with JWT



---



\## 🛠 Tech Stack



\### Frontend



\* React.js (Vite)

\* Context API

\* Custom UI Components



\### Backend



\* Node.js

\* Express.js

\* MongoDB



\### AI \& External APIs



\* Gemini API (question generation)

\* AssemblyAI (speech-to-text)

\* Murf AI (text-to-speech)

\## ⚙️ Installation \& Setup



\### 1. Clone the repository



```

git clone https://github.com/vagesh10/ai-mock-interview-assistant.git

cd ai-mock-interview-assistant

```



---



\### 2. Setup Backend



```

cd server

npm install

npm start

```



---



\### 3. Setup Frontend



```

cd client

npm install

npm run dev

```



---



\## 🔐 Environment Variables



Create a `.env` file inside the \*\*server\*\* folder and add:



```

MONGO\\\_URI=your\\\_mongodb\\\_connection

JWT\\\_SECRET=your\\\_jwt\\\_secret



GEMINI\\\_API\\\_KEY=your\\\_gemini\\\_api\\\_key

ASSEMBLYAI\\\_API\\\_KEY=your\\\_assemblyai\\\_key

MURF\\\_API\\\_KEY=your\\\_murf\\\_key

```



⚠️ Never push your `.env` file to GitHub.
